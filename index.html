a personal activity tracker for Claude Code

Track what you're doing throughout the day:

"started working on the API refactor"

"feeling scattered, need to focus"

"remind me to follow up with Sarah"

"breakthrough on the auth flow"

It's part coach, part todo list, part guardrails on your focus. Everything stays local as simple JSON files.

Over time, you can ask bigger questions: what am I avoiding? where do I lose momentum? what was I thinking last Tuesday?

Install: review the script, then:

curl -O https://raw.githubusercontent.com/jonbell-lot23/cardenas-starter/main/install.sh chmod +x install.sh && ./install.sh

Restart Claude Code. Try /track
