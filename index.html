═══ WHAT IT IS ═══

A personal activity tracker for conversational logging.

Named after García López de Cárdenas, the first European to see the Grand Canyon in 1540. He saw the scale but couldn't comprehend it fully—which perfectly captures the feeling of looking at your own activity data.

Like Cárdenas standing at the rim, the system helps you witness the vastness of what you've actually accomplished, even when you can't fully grasp it in the moment.

═══ WHAT IT DOES ═══

Zero-friction logging to daily JSON files

Query activities by date range (--today, --week, --days N)

Goals with horizons: life → yearly → quarterly → monthly → weekly

Mood and health tracking

Claude Code skills (/track, /today, /week, /goals)

Local-first. No external dependencies. Just files.

═══ PHILOSOPHY ═══

Capture depth, not just summaries.

# BAD "Wrote document" # GOOD "Mental state: Using work momentum as self-soothing— creating control when system feels chaotic"

Good entries include mental state, strategic insights, and the "why" behind decisions.

═══ INSTALLATION ═══

⚠ Security note: Review the script before running it.

1. View the installer script:

github.com/jonbell-lot23/cardenas-starter/blob/main/install.sh

2. Verify it only creates files in ~/cmd/cardenas/ and ~/.claude/

3. Download and run:

curl -O https://raw.githubusercontent.com/jonbell-lot23/cardenas-starter/main/install.sh chmod +x install.sh ./install.sh

4. Restart Claude Code

5. Try: /track "Just installed Cárdenas"
